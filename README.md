## 什么是MouseJiggle?

简单来说,MouseJiggle就是一个鼠标自动移动工具。它会在你的电脑闲置60秒后,自动让鼠标指针轻微晃动,让系统误以为你还在使用电脑,从而避免屏幕锁定或进入休眠状态。

这个"晃动"非常轻微,肉眼几乎看不出来,完全不会影响你正常使用电脑。更厉害的是,它还有个"Zen模式"——虚拟晃动,鼠标指针根本不动,只是在系统底层发送信号,告诉电脑"我还醒着呢"。

## 如何使用MouseJiggle?

使用方法非常简单:

1. 👉 [下载安装包](https://mousejiggle.org)(Windows版本仅2.1MB,Mac版本约151MB)
